# kaggle competition: March Machine Learning Mania 2025
##　概要
　このプロジェクトは、kaggleの"March Machine Learnin Mania 2025""に参加したものです。このコンペティションでは2021年から2024年までのアメリカの大学バスケのトーナメント結果をもとに2025年の結果を予測する機械学習モデルをつくることです。このリポジトリでは分析プロセス、アルゴリズム、最終結果を記録します。

## 目的
目標：2025年のアメリカの大学バスケのトーナメント結果の予測
評価指標：Brier score（https://en.wikipedia.org/wikBrier_score）

##　使用した言語、ライブラリ等

##　データセット
　このプロジェクトでは、Kaggleに提供された過去のNCAAバスケットボールの試合結果、チームの統計情報、トーナメントの情報を使用しました。主なデータソースは以下の通りです：
- **過去の試合データ**: 各試合の結果、チームID、スコア（男子・女子トーナメント）
- **チーム統計情報**: 得点、リバウンド、アシストなどのチームパフォーマンス指標

これらのデータは、Kaggleの提供するデータセットに基づいており、複数年分のNCAAトーナメントの情報が含まれています。
